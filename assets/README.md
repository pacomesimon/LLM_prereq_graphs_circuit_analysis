# Dataset Assets

This folder contains the **graph-structured datasets** generated by the framework. 

### Structure
Files are stored in **JSONL (JSON Lines)** format. Each row represents a node in the task graph, containing:
-   `id`: Unique identifier (e.g., `r1_c1`).
-   `parent_id`: ID of the prerequisite task (`null` for roots).
-   `messages`: OpenAI-style conversation history (system/user/assistant).
-   `tools`: Definitions of available actions for that specific node.
-   `tags`: Increasing complexity labels for easier analysis.

### Usage
-   To use these datasets for evaluation, upload them to the **Cascade Evaluation** tab in the Gradio app.
-   Each file is timestamped: `dataset_YYYYMMDD_HHMMSS.jsonl`.
